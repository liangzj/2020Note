1. 排序下 笔记梳理以及练习

今日总结：
1. 谭超老师直播课的一些建议：
    + 一道算法题如果你10分钟没有任何思路，尽量去看下解题思路，再来实践，因为任何算法多是经过反复精炼再提取出来的，我们重点不是发明，也很难发明，而是去理解精髓会活用。
    + leetcode上刷题，解决提交之后，一定要去看看该题排名top前面的解题方案。
2. 王铮老师关于递归的一些建议：
    + 不要试图去一级级推导整个递归的过程，只分析一级就好，当前级的子级就当作一直条件。
    + 实在要推导，可以画二叉树来推导
